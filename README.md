# Biblical Canvas

Biblical Canvas is a modern, privacy-focused Bible verse wallpaper designer. It allows users to create stunning, shareable images from any Bible verse with deep customization options.

## 🌟 Features

- **SEO Optimized**: Fully optimized with Meta tags, Open Graph, and Twitter Cards for social sharing.
- **Privacy First**: GDPR and CCPA compliant with a built-in privacy consent system. Local-first design means your data stays on your device until you choose to share.
- **Verse Lookup**: Instant lookup across hundreds of Bible versions via integrated APIs.
- **Customization**:
  - Drag-to-position text boxes for verses and references.
  - Full typography control (size, weight, line height, spacing, color).
  - Background image search via Pexels or local file upload.
  - Image adjustments (brightness, contrast, blur, saturation).
- **One-Click Share**:
  - **Share with Developer**: Instantly upload your design and open an email draft with a public link to share your creation with the development team.
  - **Native Share**: Use your device's built-in share menu (mobile & desktop).
  - **PNG Export**: High-quality download for wallpapers.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
npm install
cd server && npm install
```

### 2. Configure Environment
Create a `.env` file in the root directory:
```env
VITE_API_BIBLE_KEY=your_key_here
VITE_PEXELS_API_KEY=your_key_here
ADMIN_PASSWORD=your_admin_password
```

### 3. Start the Application
Start the frontend and backend simultaneously:
```bash
# Start frontend (Vite)
npm run dev

# Start share server (Node.js)
npm run server
```
